## Aktivasi virtual environment di cmd

1. buat virtual environment
```bash
python -m venv .venv
```

2. aktivasi virtual environment
```bash
.venv\Scripts\activate
```

3. install semua library
```bash
pip install -r requirements.txt
```

4. running model
```bash
python train.py
```
